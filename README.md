# LionParcel
Automate Register Lion Parcel X SMS HUB

How To Install This Bot

1.) Install Node JS https://nodejs.org/
2.) Create a Folder on disk /Lion Parcel
3.) Extract Rar On /Lion Parcel
4.) Create apikey.txt (sms hub apikey )
5.) Open CMD
6.) cd c:\location bot
7.) npm i
